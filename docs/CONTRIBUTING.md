# Contributing to Soflution LLM

## Ways to Contribute

1. **Run benchmarks**: `python3 scripts/self_improve.py` then submit a PR
2. **Improve Modelfile**: better parameters = better model for everyone
3. **Add fine-tunes**: specialize the model for your domain in `fine-tunes/`
4. **Improve the agent**: make `scripts/self_improve.py` smarter

## PR Guidelines
- One thing per PR
- Include hardware specs in benchmark PRs
- Apache 2.0 license applies to all contributions
